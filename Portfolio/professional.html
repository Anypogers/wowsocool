Sobre os Projetos

Discord Bot

O Bot de Discord foi feito usando NodeJS, utilizando o pacote de API do Discord para JS (DiscordJS). Juntamente com o pacote Better-SQLite para uso com um banco de dados em SQLite.

Para outros tipos de dados como por exemplo itens que os usuarios podem coletar, ele utiliza arquivos de tipo JSON para armazenamento de informacao.

O bot contem varias funcoes divertidas para interacao no servidor, como por exemplo, poder jogar pedra papel tesoura contra outros usuarios.
